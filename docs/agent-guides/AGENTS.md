# AGENTS dla `docs/agent-guides/`

Ten katalog zawiera lokalne przewodniki dla agentow pracujacych nad projektem.

- Przewodniki maja byc krotkie, praktyczne i powiazane z realnymi plikami repo.
- Linkuj oficjalne zrodla prawdy zamiast kopiowac dlugie fragmenty dokumentacji zewnetrznej.
- Nie dodawaj ogolnych porad, jesli nie przekladaja sie na decyzje w tym rozszerzeniu.
- Aktualizuj przewodnik, gdy zmienia sie architektura projektu albo proces walidacji.
