# Dokumentacja

Ten katalog zawiera notatki techniczne i operacyjne dla rozszerzenia Chrome.

1. [Mapa systemu](system-map.md) opisuje komponenty rozszerzenia i lokalny przepływ builda.
2. [Kontrakty](contracts/) opisują trwałe zasady projektu i ustalenia współdzielone z backendem Meet2Note.
3. [Runbooki](runbooks/) zawierają powtarzalne procedury walidacyjne.
4. [Specyfikacje](specs/) są miejscem na robocze specyfikacje przyszłych większych zmian.
5. [Przewodniki agentowe](agent-guides/) zawierają lokalne zasady pracy agentów nad Chrome Extension i TypeScript.

Projekt nie używa obecnie osobnych dokumentów deploymentu ani projektowych umiejętności agentowych poza przewodnikami w `agent-guides/`.
